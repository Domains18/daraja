# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-13

### Added
- Initial release of Daraja SDK
- Type-safe TypeScript implementation
- Environment-specific clients (`daraja.sandbox` and `daraja.production`)
- STK Push (Lipa Na M-Pesa Online) support
- STK Push query support for transaction status checking
- Automatic access token caching for improved performance
- Comprehensive TypeScript type definitions
- Detailed error handling with descriptive messages
- Full documentation and usage examples

### Features
- Fluent API design for better developer experience
- Support for both sandbox and production environments
- Configurable timeout settings
- Automatic password generation for M-Pesa requests
- ISO 8601 timestamp formatting

### Developer Experience
- Complete TypeScript support with exported types
- IntelliSense support in modern IDEs
- Example usage files
- Comprehensive README documentation
- Environment variable examples
